# Streamflow 🚧 Work in Progress

The frontend for the Task Manager app, built with Next.js and Tailwind CSS.

> **Status**: Early scaffolding — UI and structure are actively being built out.

---

## Tech Stack

- **Framework** — Next.js 14 (App Router)
- **Styling** — Tailwind CSS
- **Data Fetching** — Native Fetch API
- **Backend** — Express + PostgreSQL (via Task Manager API)

> **Planned**: Migrating backend to Supabase for public deployment.

---

## Notes

Currently consumes the local Express + PostgreSQL API. Once ready for public deployment, the backend will be replaced with Supabase.
